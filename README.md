# Quoin Buildings Website Starter

This folder is set up with Vite so you can build a website using plain HTML, CSS, and JavaScript.

## Scripts

- `npm install` installs dependencies.
- `npm run dev` starts the local development server.
- `npm run build` creates a production build in `dist/`.
- `npm run preview` previews the production build locally.

## Main files

- `index.html` is the document shell.
- `src/main.js` contains the page markup.
- `src/style.css` contains the site styles.
